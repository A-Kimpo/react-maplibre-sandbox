## Решённые вопросы
| Вопрос | Решение |
|---|---|
| Next.js или Vite? | Next.js — нужен SEO (B2C) |
| Карта: Mapbox или MapLibre? | MapLibre — open-source, без токена |
| Тайлы? | OpenStreetMap (начало), позже — кастомные |
| VR: что первым? | Panorama equirectangular (как Google Maps), затем 3D сцены |
| Архитектура? | FSD + изолированная map library в shared/lib/map |
| Данные? | REST API, сейчас mock → потом подключим эндпоинты |
